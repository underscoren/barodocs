About Barodocs
This is an experimental documentation web app for Barotrauma.

Barodocs is made by scanning the XML definitions from the retail game, and then dynamically generating the pages from the contents.

Currently, only a limited portion of the game's XML is scanned and displayed. Items and Afflictions are mostly mapped out, as well as relations between them (e.g. made by, caused by). Afflictions also use graphs to display complex behaviour of effects, if needed (Tip: you can click on the labels to hide or show them).

You can search by identifier or name, and also use prefixes for complex search operations:

  • @ to search by type (e.g. @affliction or @item)
  • # to search by item tags (e.g. #gun)
  • : to search by item category or affliction type (e.g. :damage)

You can also press Tab to open up the sidebar.

Source code is available here, feel free to submit any issues, make feature requests or contribute PRs.
